# User Stories - Capstone Mobile Application

### 1. User Registration (Sign-up)
As a new user, I want to create an account using my username, email, and password so that I can have a personalized experience.

### 2. User Authentication (Login)
As a returning user, I want to log in securely with my email and password so that I can access my saved data.

### 3. Home Screen Feed
As an active user, I want to view a clean home dashboard containing a customized logo and a list of meditation exercises.

### 4. Viewing Detailed Content
As a user browsing the app, I want to tap on a specific item to open a detailed view with complete descriptions.

### 5. Managing Favorites (Local Storage)
As a user, I want to save specific exercises to my favorites list locally using SharedPreferences so that I can quickly access them.

### 6. Dynamic Daily Inspiration (API Integration)
As a user looking for daily motivation, I want the app to fetch and display a dynamic quote from an external API.

### 7. Customizing App Appearance (Theme Settings)
As a user, I want to toggle between light and dark modes in the settings menu.

### 8. Accessing Settings Menu
As a user navigating the app, I want a dedicated settings icon on my dashboard that opens a menu of configurations.

### 9. Scheduling Reminders (Local Notifications)
As a user, I want to schedule custom reminder alerts on my device so that I never miss my daily session.
